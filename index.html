Diferentes formas de salida de texto en JavaScript

En esta página se muestran ejemplos usando datos personales de Adrian Tadeo Montes Vazquez. Se utilizan innerHTML, innerText, document.write(), alert(), console.log() y print().

Salida con innerHTML

Inserta contenido HTML dentro de un elemento del documento.

Salida con innerText

Muestra texto plano dentro de un elemento, sin interpretar etiquetas HTML.

Salida con document.write()

Escribe contenido directamente en el documento cuando la página se está cargando.

Salida con alert()

Muestra un cuadro emergente con información del usuario.

Salida en consola

Envía información a la consola del navegador para depuración o seguimiento.

Salida con print()

Abre el cuadro de impresión del navegador para imprimir la página.
